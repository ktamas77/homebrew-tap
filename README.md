# ktamas77/homebrew-tap

Homebrew formulae for [@ktamas77](https://github.com/ktamas77)'s tools.

## agentop

`top`, but for your running coding agents — Claude Code, Codex, Grok, Gemini, and
Antigravity. See [ktamas77/agentop](https://github.com/ktamas77/agentop).

```sh
brew install ktamas77/tap/agentop
```

Then run:

```sh
agentop          # live dashboard
agentop --demo   # preview with sample data
```
